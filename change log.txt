MAZE API Change Log

1.2.0.0 (2020 03)
Migrated codebase to github
Added support for getting object state, maze state, elapsed time, current maze name

1.1.0.0 (2017 11)
Added support to disable Gravity via API
Added support to Get and Set model position, rotation, scale, ID and name

1.0.0.1 (2017 03)
Added set/reset move speed option
	MazeCOMM include SetMoveSpeed function
Added support for compiling sample application mazeCommBridge without C++ std lib
